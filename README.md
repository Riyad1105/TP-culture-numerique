\## Description

Ce projet contient les algorithmes développés dans le cadre du TP de culture numérique.



\## Comment ouvrir les fichiers



1\. Installer AlgoBox depuis \[le site officiel](http://www.xm1math.net/algobox/)

2\. Ouvrir AlgoBox

3\. Fichier → Ouvrir → Sélectionner le fichier `.alg` dans le dossier `src/`
